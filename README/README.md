# face-recognition

> 人识别云(认证，检索，熟悉，场景)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run serve

# build for production and view the bundle analyzer report
npm run build:prod
```

### 系统操作流程：
1.创建设备
2.创建通行规则
3.创建人员
3.从设备下发人员
4.将人员添加至通行规则
5.查人员对应的设备及通行规则
